# MDbook

Markdown documentation generation tool.

You can install this script by running the following command in your terminal.

```sh
curl -sL https://MR-Addict.github.io/setup/api/script/mdbook | bash
```

Below is the raw script content, which you can also get from the command above from the api https://MR-Addict.github.io/setup/api/script/mdbook.

```sh
#! /bin/bash

# 1. Install mdbook
if ! command -v mdbook &>/dev/null ; then
  echo "[INFO] Installing mdbook..."
  wget -q https://github.com/rust-lang/mdBook/releases/download/v0.4.48/mdbook-v0.4.48-x86_64-unknown-linux-gnu.tar.gz -O mdbook.tar.gz
  tar -zxf mdbook.tar.gz
  rm mdbook.tar.gz
  chmod u+x mdbook
  sudo mv mdbook /usr/local/bin
else
  echo "[WARN] You have already installed mdbook!"
fi

else

fi
